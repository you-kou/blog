# ExampleMatcher.GenericPropertyMatcher

```java
public static class ExampleMatcher.GenericPropertyMatcher
extends Object
```

一个通用的属性匹配器，用于指定字符串匹配方式和大小写敏感性。

## 方法

### contains

```java
public ExampleMatcher.GenericPropertyMatcher contains()
```
